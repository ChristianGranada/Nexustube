# Proyecto NexusTube - Sistema de Login y UI

Este proyecto es una aplicación web desarrollada como práctica de frontend, enfocada en la creación de una interfaz tipo login y manejo básico de datos en el navegador.

---

## Demo en vivo
Puedes ver el proyecto funcionando aquí:

https://christiangranada.github.io/Nexustube/

---

## Descripción

El proyecto consiste en una página web con interfaz moderna donde se implementa un sistema básico de login utilizando tecnologías del frontend. También se hace uso de almacenamiento local del navegador para persistir información del usuario.

---

## Funcionalidades

- Interfaz de login
- Validación básica de formulario
- Uso de `localStorage` para guardar datos del usuario
- Redirección entre páginas
- Diseño responsivo con Bootstrap

---

## Tecnologías utilizadas

- HTML5
- CSS3
- Bootstrap
- JavaScript (Vanilla JS)
- Git & GitHub Pages
